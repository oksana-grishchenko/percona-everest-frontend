# Install bit

```bash
make install-bit
```

# Install

```bash
make inisdfsdfst
```

# Running Bit with

```bash
bit start
```

# Running a particular app (e.g. Everest)

```bash
cd apps/<app-name>
make run
```

# Run unit tests for app
```bash
cd apps/<app-name>
make test
```

# Run E2E tests for app
```bash
cd apps/<app-name>
make test-ui
```

# Build app
```bash
cd apps/<app-name>
make build
```
